Automount Windows drive in Linux

December 24, 2023 One-minute read

Find out where the windows drive is located:

lsblk

Edit fstab file:

sudo nvim /etc/fstab

For example, sdb2:

# <file system>             <mount point>  <type>  <options>  <dump>  <pass>```
/dev/sdb2                      /media/data       ntfs    defaults 0 0

Reboot the system:

reboot
