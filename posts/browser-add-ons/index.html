Bypass Paywalls Clean Link to heading

Read articles from sites like Medium. Not available in plugin stores, need to download from GitHub and import it. For e.g. .xpi file for Firefox.

Dark Reader Link to heading

Enable dark mode for any site.

Default toggle keybind: alt + shift + d

Video Speed Controller Link to heading

Hotkeys for changing playback speed on most video platforms, useful for Netflix/YouTube etc.

Change increment/decrement steps in the settings to 0.5.

Default keybindings:

  • d (speed up)
  • s (slow down)
  • v (hide/show the current speed)
  • r (reset to normal speed)
