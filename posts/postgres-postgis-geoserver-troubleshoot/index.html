Postgres - PostGIS - Geoserver setup troubleshoot

May 15, 2024 One-minute read

Postgres Link to heading

After downloading postgresql, login with psql

psql -h localhost -U postgres

SHOW hba_file;

The command will reveal a path to a file, edit that file with sudo:

sudo -i

In the file, replace all the peer/md5 column with trust. Reference https://stackoverflow.com/questions/18664074/getting-error-peer-authentication-failed-for-user-postgres-when-trying-to-ge

PostGIS Link to heading

sudo apt install postgis postgresql-postgis

Run in PGAdmin or command line, run this query before importing any data: CREATE EXTENSION postgis;

compute bounding box not working in geoserver: restart postgresql:

sudo systemctl restart postgresql

layer preview, clicking openlayers download wms instead of opening in portal.
